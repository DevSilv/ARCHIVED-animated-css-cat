Simple animated CSS cat

About the website and the author

This website is intended to present an animated cat.

To see the cat, click any of the links in the main menu with the word "cat".

I (me, silvuss) am the author of this website. This website is published by courtesy of the GitHub Pages hosting. You can find sources of this website in its repository on my GitHub account.

About the cat

The cat is animated only using the CSS language in 3 ways:

  • using only CSS;
  • using the Sass language (with the SCSS syntax), compiled then to CSS;
  • using the LESS language, compiled then to CSS.

The Sass and LESS versions differ in syntactic sugar. You can find the broader descriptions which features of these languages are used in the README.

There are two animations that the cat is expected to show: moving its tail and cleaning its paws. The tail shall move every 5 seconds, and both the paws shall be cleaned every 20 seconds.

I am not an expert on cats. When I was creating the tail animation, unfortunately, I forgot to look for any information of the behavior of cats. Therefore, the moves that the cat is making with its tail are not to be strictly classified; I think that they may seem to indicate something like playful mood (according to this Wikipedia's article on cat).
